# ALTFaze Freelancer System - Complete User Guide

## 🎯 Overview

The ALTFaze Freelancer System allows you to:
- **Create and sell templates** - Build and monetize design/code templates
- **Bid on client projects** - Find and bid on work from clients
- **Track earnings** - Monitor sales, income, and transactions
- **Manage your portfolio** - Edit, update, and delete templates

---

## 📝 Template Management

### Create a New Template

1. **Navigate:** Click "Create Template" button on dashboard
2. **Fill Form:**
   - **Title** - Name your template
   - **Description** - Detailed explanation of what's included
   - **Price (USD)** - Set your price
   - **Category** - Select: Web Design, E-commerce, Blog, Portfolio, SaaS, Landing Page, Mobile App, Other
   - **Preview Image URL** - Link to template preview image
   - **Template File URL** - Link to downloadable template file

3. **Technology Stack** - Search and add technologies:
   - React, Next.js, Vue, Angular, Tailwind CSS, Bootstrap, TypeScript, JavaScript, Node.js, MongoDB, PostgreSQL, GraphQL, Stripe, AWS
   - Type to search → Click "+ Technology" to add

4. **Key Features** - Add features as tags:
   - Type feature name (e.g., "Responsive Design")
   - Press Enter or click "Add"
   - Remove by clicking X on tag

5. **Submit** - Click "Save Changes" to create

### Edit an Existing Template

1. **Navigate:** Dashboard → Templates tab → Click "Edit" button
2. **Update Fields:** All sections work the same as creation
3. **Save Changes** - Updates immediately

### Delete a Template

1. **Navigate:** Dashboard → Templates tab
2. **Click Delete** button on the template
3. **Confirm** in the dialog that appears
4. **Note:** Cannot delete templates with existing purchases

## 💼 Project Bidding System

### Browse Available Projects

1. **Navigate:** Click "Browse Projects" from dashboard or go to `/projects`
2. **Search:** Use search bar to find specific projects
3. **Filter by Category:**
   - Web Design, Web Development, Mobile App, UI/UX Design, Graphic Design, Content Writing, SEO, Marketing

4. **Filter by Budget:**
   - All Budgets, $0-100, $100-500, $500-1000, $1000+

5. **View Projects:** See title, description, required skills, budget, and bid count

### Submit a Bid

1. **From Projects List:** Click "View & Bid" button
2. **Project Detail Page:**
   - Review project description and requirements
   - See existing bids from other freelancers
   - View budget, deadline, and required skills

3. **Bid Form (Right Sidebar):**
   - **Your Bid Amount** - Enter your proposed price
   - **Delivery Time (Days)** - How many days to complete
   - **Proposal Message** - Explain why you're a good fit

4. **Submit Bid** - Click "Submit Bid" button
   - If updating existing bid, it shows "updated" message
   - View updated bid in project page

### View Your Bids

1. **Navigate:** Click "My Bids" from dashboard or go to `/freelancer/my-bids`
2. **See All Bids:**
   - Project title and creator name
   - Your bid amount vs project budget
   - Delivery timeline
   - Your proposal text
   - Required skills

3. **Actions:**
   - "View Project" - Go back to project page
   - "Edit Bid" - Update your bid details

## 💰 Wallet & Payments

### How Earnings Work

1. **Client Purchases Template** → Funds go to your wallet
2. **View Balance** - Dashboard shows current wallet balance
3. **Withdraw Funds** - Request transfer to your bank account
4. **Track History** - All transactions visible in Earnings page

### Payment Status
- **Pending** - Recently earned, processing
- **Completed** - Confirmed and in your account
- **Withdrawn** - Transferred to your bank

## 🆘 Troubleshooting

### Issues & Solutions

**Can't edit template**
- Only you can edit your own templates
- Make sure you're logged in as the template creator

**Bid won't submit**
- Ensure all fields are filled (amount, days, message)
- Check bid amount is a valid number

**Can't see earnings**
- Navigate to Earnings page to see full details
- Dashboard shows summary only

**Delete button disabled**
- Templates with purchases cannot be deleted
- You can still hide or stop selling them

**Profile not updating**
- Check your internet connection
- Try refreshing the page
- Contact support if issue persists
